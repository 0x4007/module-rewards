To view GitHub content, please provide a personal access token with 'repo' scope access.

Invalid token format. Please check your token and try again.

How to create a token:

  1. Go to GitHub Token Settings
  2. Click "Generate new token" > "Generate new token (classic)"
  3. Add a note like "GitHub Content Viewer"
  4. Select the "repo" scope
  5. Click "Generate token"
  6. Copy the generated token and paste it above

Note: Your token is stored only in your browser's local storage and is never sent to our servers.
